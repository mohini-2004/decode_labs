# 🐍 Python Mini Projects

This repository contains beginner-to-intermediate level Python projects developed as part of Python programming practice and industrial training. These projects focus on core programming concepts such as lists, loops, functions, dictionaries, accumulators, and exception handling.

---

# 📌 Project 1: To-Do List Application

## Overview

The To-Do List Application helps users manage daily tasks through a simple command-line interface. Users can add tasks, view tasks, mark tasks as completed, and delete tasks.

## Features

- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Menu-driven interface
- Input validation and error handling

## Concepts Used

- Lists
- Dictionaries
- Functions
- Loops
- Conditional Statements
- Exception Handling

---

# 📌 Project 2: Expense Tracker

## Overview

The Expense Tracker is a command-line application that allows users to record and manage their expenses. The program calculates total spending and average expenses using Python's mathematical and data-processing capabilities.

## Features

- Add expenses
- View expense history
- Calculate total spending
- Calculate average expense
- Delete expenses
- Menu-driven interface
- Error handling

## Concepts Used

- Lists
- Functions
- Loops
- Mathematical Operations
- Accumulators
- Exception Handling
